🚗 Carsharing DLM System — Automated Backup & Watchdog ServiceАвтономная, изолированная отказоустойчивая система резервного копирования и непрерывного мониторинга (Watchdog) для инфраструктуры каршеринга. Система спроектирована на базе СУБД PostgreSQL 17 с использованием механизмов нативной физической инкрементальной репликации и распределенной очереди задач Celery + Redis.
